Random Facts API Documentation

Welcome to the Random Facts API! Use the endpoints below to fetch random facts or submit your own. All endpoints are prefixed with /api, so base URLs will look like https://mrboys.vip/api.

1. Fetch a Random Fact

Endpoint: GET /api/facts/random

Description: Retrieves a random fact from the database.

Query Parameters:

  • language (optional): Specify the language of the fact (e.g., en for English). Defaults to English.

Example Request:

GET https://mrboys.vip/api/facts/random?language=en

Response:

{
  "fact": "Honey never spoils.",
  "category": "Science",
  "language": "en",
  "source": "user"
}

2. Submit a New Fact

Endpoint: POST /api/facts

Description: Allows users to submit a new fact to the database.

Request Body: JSON object containing:

  • fact (required): The fact content, as a string.
  • category (required): The category of the fact (e.g., "Science").
  • language (optional): Language code for the fact (e.g., en for English). Defaults to English.
  • source (optional): Source of the fact. Defaults to user.

Example Request:

POST https://mrboys.vip/api/facts 
{
  "fact": "A group of flamingos is called a 'flamboyance'.",
  "category": "Animals",
  "language": "en"
}

Response:

{
  "_id": "61234abcde1234567890",
  "fact": "A group of flamingos is called a 'flamboyance'.",
  "category": "Animals",
  "language": "en",
  "source": "user"
}
